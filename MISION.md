# PROYECTO: ARTICULISTA (Affiliate Niche Site)

## OBJETIVO DE NEGOCIO
Crear una plataforma web minimalista y ultra-rápida optimizada para SEO y marketing de afiliados. El objetivo final es la monetización mediante enlaces de referidos (Amazon, etc.).

## REQUISITOS TÉCNICOS (Tech Stack)
- HTML5 Semántico (Crucial para SEO).
- CSS Moderno (Flexbox/Grid), diseño Mobile-First.
- NO usar frameworks pesados (React/Vue) innecesarios. Usar Vanilla JS si se requiere interactividad.
- Performance: Google Lighthouse Score > 95.

## ESTRUCTURA DESEADA
1. **Home:** Listado de artículos destacados con "Grid Layout".
2. **Post Template:** Diseño de artículo con zona para "Call to Action" (Botones de compra) bien visibles.
3. **Sidebar/Footer:** Zonas para disclaimer legal y enlaces de afiliado.

## ESTILO
- Tipografía limpia (Sans Serif para UI, Serif para lectura).
- Espacio en blanco generoso (White space) para facilitar la lectura.
- Paleta de colores: Confianza (Azules/Grises) con acentos de conversión (Naranja/Verde).
