# Project Wiki

## Glossary
- **Epic**: Large initiative (e.g., "Visual Engagement").
- **Feature**: New user-facing functionality.
- **Story point**: Complexity estimation (1=Easy, 3=Medium, 5=Hard).
- **Spike**: Research task.
- **Tech debt**: Compromises made for speed.

## Definition of Done (DoD)
1. **Code:** Clean, commented, linted.
2. **Tests:** Manual verification passed (simulated), no regressions.
3. **Docs:** Changelog updated.
4. **Visuals:** Responsive on Mobile (375px) and Desktop.
